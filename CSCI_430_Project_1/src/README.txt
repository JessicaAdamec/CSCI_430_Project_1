To run the project:
1) Navigate to project folder (e.g. C:\Project2Stage2)
2) To compile, enter the following command: javac WarehouseContext.java
OR IF AN ERROR APPEARS
javac WarehouseContext.java -Xlint:unchecked
3) To run: java WarehouseContext

This warehouse system manages clients, products, and suppliers. Clients 
may put products into a shopping cart.
