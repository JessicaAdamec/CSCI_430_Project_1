To run the project:
1) Navigate to project folder (e.g. C:\ProjectCodeFolder)
2) To compile, enter the following command: javac UserInterface.java
3) To run: java UserInterface

This warehouse system manages clients, products, and suppliers. Clients 
may put products into a shopping cart.
